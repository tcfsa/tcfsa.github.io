Exercice 4 — Décodage d’adresse (RAM 4 Ko)

Objectif : trouver l’état de A15..A12, la fonction de décodage, puis la plage d’adresses

Énoncé + lecture du schéma

Une RAM de 4 Ko est connectée : A0..A11 vont à la mémoire. Les bits A12..A15 servent au décodage via une porte logique vers /CS.

RAM 4 Ko ⇒ 12 lignes d’adresse internes

Ce qu’on lit sur le câblage

  • La mémoire a A0..A11 → donc 2¹² = 4096 octets = 4 Ko.
  • /CS est actif à 0 (barre sur CS).
  • Le décodage utilise A12..A15 avec des inverseurs + une porte NAND (bulle en sortie).

Rappel : une NAND sort 0 uniquement si toutes ses entrées sont à 1.

Schéma simplifié (lisible & responsive)

Q1 — État de A12..A15 et fonction de décodage

/CS est actif bas : on cherche les bits qui rendent la sortie de la NAND = 0.

Conclusion : A15 A14 A13 A12 = 0 0 1 0

1) Condition de sélection

La porte est une NAND (bulle en sortie). Donc la sortie vaut 0 seulement si toutes les entrées valent 1.

  • Entrée 1 = ¬A15 → pour avoir 1 ⇒ A15=0
  • Entrée 2 = ¬A14 → pour avoir 1 ⇒ A14=0
  • Entrée 3 = A13 → pour avoir 1 ⇒ A13=1
  • Entrée 4 = ¬A12 → pour avoir 1 ⇒ A12=0

État requis

A15 A14 A13 A12 = 0 0 1 0  ⇒  0010₂ = 2₁₆

2) Fonction de décodage

La sortie de la NAND (connectée à /CS) vaut :

/CS = ¬( ¬A15 · ¬A14 · A13 · ¬A12 )

Forme “OU” (De Morgan), souvent plus simple à lire :

/CS = A15 + A14 + ¬A13 + A12

Donc /CS=0 uniquement quand A15=0, A14=0, A13=1, A12=0.

Q2 — Plage d’adresses occupée par la RAM (hex)

RAM 4 Ko ⇒ A0..A11 varient (000..FFF). A15..A12 fixés à 0010.

Plage : 0x2000 → 0x2FFF

Étapes rapides

  • Taille RAM = 4 Ko = 4096 = 2¹² adresses → 12 bits internes.
  • Donc A11..A0 parcourent : 0x000 → 0xFFF.
  • Et on a trouvé : A15..A12 = 0010₂ (nibble = 0x2).

Plage d’adresses

Début : 0x2000
Fin : 0x2FFF

Rappel équation

équivalent : /CS = A15 + A14 + ¬A13 + A12

Lecture automatique de la plage d’adresses

Avec une RAM 4 Ko, l’offset interne est toujours 0x000 → 0xFFF. Le nibble A15..A12 choisit le bloc de 4 Ko.

Quand A15..A12 = 0x2, la RAM répond sur toutes les adresses 0x2xxx.
